Главная

Краткий справочник по командам

  • ls — показывает список каталогов и файлов в текущем каталоге;
  • cd — показывает имя текущего каталога или выполняет смену текущего каталога;
  • pwd — показывает путь к текущему каталогу
  • file — показывает тип файла;
  • mv — переименовывает или перемещает файл или каталог;
  • rm — удаляет файл;
  • rmdir - удаляет каталог;
  • mkdir — создает каталог;
  • cat — показывает содержимое файла
